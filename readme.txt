1.    Copy Real Performance BenignSource in your plugin directory
2.    In your WordPress administration, go to the Plugins page
3.    Activate the Real Performance BenignSource plugin and a subpage for the plugin will appear in your Manage menu.
4.    Go to the Settings >Real Performance BenignSource and make the settings step by step
